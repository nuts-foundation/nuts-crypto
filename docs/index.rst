.. _nuts-crypto:

Welcome to Nuts Crypto's documentation!
=======================================

The Nuts crypto project can be used as a library for doing Nuts specific crypto actions.
Because it has a configurable backend, you can use a file storage backend for local development and use something like Vault in production.
